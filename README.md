# FTB University

이 한국어 번역은 FTB University의 한국어 번역입니다. 

이번 번역에서는 University 에서 제공하는 기본언어가 번역이됩니다.

각 모드의 번역은 추후 따로 프로젝트를 개설하겠습니다.

이 번역은 국내 커뮤니티 디시인사이드의 번역을 사용하고있습니다.
